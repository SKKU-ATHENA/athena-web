## LLM이란?

**Large Language Model(대규모 언어 모델)**은 방대한 텍스트 데이터로 훈련된 딥러닝 모델입니다.
Transformer 아키텍처를 기반으로 하며, 텍스트 생성, 요약, 번역, 추론 등 다양한 언어 작업을 수행합니다.

## 주요 모델 비교

| 모델 | 개발사 | 특징 | ATHENA 용도 |
|------|--------|------|------------|
| GPT-4o | OpenAI | 범용, 높은 성능 | 평가/데모 |
| GPT-4o-mini | OpenAI | 가성비, 빠른 응답 | 개발 중 테스트 |

| Llama 3 | Meta | 오픈소스, 로컬 실행 가능 | 로컬 개발 |

## ATHENA에서의 활용

### 1. 엔티티/관계 추출
LLM에게 텍스트를 주고, 구조화된 엔티티(사람, 개념, 이벤트)와 관계를 추출합니다.

```python
prompt = """
다음 텍스트에서 엔티티와 관계를 추출하세요.

텍스트: {text}

출력 형식:
- 엔티티: (이름, 타입)
- 관계: (주체, 관계, 객체)
"""
```

### 2. 인과 관계 분석
"왜?" 질문에 대한 인과 사슬을 생성합니다.

### 3. 자연어 쿼리 → Cypher 변환
사용자의 자연어 질문을 Neo4j Cypher 쿼리로 변환합니다.
