## ATHENA 학습 웹사이트에 오신 것을 환영합니다

이 웹사이트는 ATHENA 프로젝트 팀원들의 학습 내용을 정리하고 공유하는 공간입니다.

### 글 작성 방법

1. `content/` 디렉토리에 `.md` 파일을 생성합니다
2. 파일 상단에 frontmatter를 작성합니다:

```yaml
---
title: "글 제목"
date: "2026-03-21"
author: "작성자"
description: "간단한 설명"
---
```

3. 본문을 마크다운으로 작성합니다
4. PR을 올리면 자동으로 배포됩니다

### 기술 스택

- **Next.js** — React 기반 프레임워크
- **Tailwind CSS** — 스타일링
- **GitHub Pages** — 정적 호스팅
